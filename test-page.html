Edge Cases should replace

Empty colon pair (should NOT match): :: or : :

Single colon: just a : in the middle

Triple colon: :::404::: — only the inner :404: should match

Back-to-back: :1up::404::500: — three emojis, no spaces

Nested-looking: :absolute:cinema: — matches :absolute: if it exists, then leftover

Non-existent emoji: :this_emoji_definitely_does_not_exist_xyz: — should stay as text

Code Block — Should NOT Replace should skip

The extension skips <code> and <pre> tags:

const status = ":404:"; // should stay as text 
function getEmoji() {
  return ":absolute_cinema:";  // should stay as text
  // :alarm: :alert: :1up:
}

But this paragraph after the code block :1up: should be replaced.
